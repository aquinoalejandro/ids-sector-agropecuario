## Resumen del trabajo práctico

Este trabajo práctico investiga el comportamiento de las heladas en el sector agropecuario a partir de datos climáticos.

Se realizó un análisis exploratorio de variables meteorológicas, se calcularon estadísticas descriptivas y se estudiaron patrones de ocurrencia de heladas. El objetivo es comprender cómo se distribuyen las heladas a lo largo del año y qué variables están más relacionadas con su aparición.

### Puntos clave

- Análisis exploratorio de datos climáticos.
- Estadísticas descriptivas para variables relevantes.
- Evaluación de la frecuencia y distribución de heladas.
- Identificación de correlaciones entre variables meteorológicas.

### Resultados principales

- Se describe el comportamiento mensual de las heladas.
- Se resumen las medidas de tendencia central y dispersión de las variables.
- Se documenta la relación entre temperatura, humedad, presión y heladas.
